**Dla MySQL/MariaDB:**

```sql
SET SQL_LOG_BIN = 0;
DROP TABLE Użytkownicy;
SET SQL_LOG_BIN = 1;
```

**Dla PostgreSQL:**

W PostgreSQL nie ma bezpośredniej możliwości wyłączenia logowania dla pojedynczej operacji. Można jednak użyć transakcji, aby ograniczyć wpływ na logi, choć nadal będzie widoczna sama transakcja:

```sql
BEGIN;
SET log_destination = 'stderr';
SET client_min_messages = 'warning';
DROP TABLE Użytkownicy;
ROLLBACK;
```

```sql
BEGIN;
SET log_destination = 'stderr';
SET client_min_messages = 'warning';
DROP TABLE Użytkownicy;
COMMIT;
```

**Dla Microsoft SQL Server:**

W Microsoft SQL Server można użyć trybu TRUNCATE_ONLY, aby uniknąć logowania:

```sql
ALTER TABLE Użytkownicy SET (SYSTEM_VERSIONING = OFF);
TRUNCATE TABLE Użytkownicy;
DROP TABLE Użytkownicy;
```

```sql
TRUNCATE TABLE Użytkownicy;
DROP TABLE Użytkownicy;
```
